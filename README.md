# Pip
Garmin Watchface inspired by the Pip-Boy 2000 Mk VI from [Fallout](https://en.wikipedia.org/wiki/Fallout_(franchise)).

![alt text](https://raw.githubusercontent.com/quinnshultz/pip/main/resources/screenshot.png "A screenshot of the watch face on Descent G1 simulator.")

## Features
- Time
- Date
- Steps
- Heartrate
- Weather
- Sunrise/Sunset
